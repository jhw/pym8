# Overview

A Python package for reading and writing Dirtywave M8 files (`.m8s`, `.m8i`)

https://dirtywave.com/

# Latest supported file format 

- 4.0.33 [OS 5.0.1]

# Features

- Songs, chains
- Phrases, FX
- Instruments
  - WavSynth
  - MacroSynth
  - Sampler
  - FMSynth
  - HyperSynth
- Modulators

# Roadmap
