# Overview

A Python package for reading and writing Dirtywave M8 files

https://dirtywave.com/

# Current version supported

4.0.1

# Supported

- Songs, chains, phrases, FX
- Instruments
  - WavSynth
  - MacroSynth
  - Sampler
  - FMSynth
  - HyperSynth
- m8s, m8i files
