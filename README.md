# Overview

A Python package for reading and writing Dirtywave M8 files

https://dirtywave.com/

# Supported versions

- OS 4.0.1 / file format 4.0.1

# Features

- Songs, chains
- Phrases, FX
- Instruments
  - WavSynth
  - MacroSynth
  - Sampler
  - FMSynth
  - HyperSynth
- Modulators
- m8s, m8i files
